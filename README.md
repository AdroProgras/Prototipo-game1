# Asymmetric Inversion (Temporary Title)

An asymmetric binary puzzle game built in C# and targeting Unity. The mechanics revolve around a custom-shaped grid where players interact with nodes under strict state rules.

## 🚀 Current Status
- **Core Engine:** Conceptualized via formal GDD.
- **Prototypes:** C# Console application with basic row-inversion rules completed.

## 📄 Game Design Document (GDD)
The core design rules, grid geometry definitions, and algorithmic specs (BFS Inverse Generation Engine) are located in the [docs/GDD.md](docs/GDD.md) file.
